# Pomodoro Web App Features

## 必須機能（MVP）
- タイマーの開始 / 一時停止 / リセット
- 作業 / 短休憩 / 長休憩モード切替
- カウントダウン表示（mm:ss）
- モードごとのデフォルト時間（例: 25/5/15）
- サイクル管理（例: 作業4回ごとに長休憩）
- 0秒到達時の自動遷移
- UI状態の反映（ボタン状態、ラベル更新）

## 履歴・可視化（必須）
- 履歴と可視化のためのダッシュボード
- 作業回数や合計時間の集計表示

## アクセシビリティ（必須）
- マウスクリックで開始/停止が可能
- キーボード操作: `S` で開始、`I` で停止、`E` で終了

## 使い勝手のための基本機能
- 設定変更（作業/短休憩/長休憩時間、長休憩までの回数）
- 設定の保存（localStorage）
- 最後の状態の復元（再読み込み時）
- モード切替時の視覚的な強調

## 通知・フィードバック（任意）
- 終了時の音通知
- ブラウザ通知（許可時）
- タイトル更新（残り時間表示）

## アクセシビリティ・品質（推奨）
- キーボード操作のフォーカス可視化
- 画面サイズ対応（モバイル/デスクトップ）
- コントラスト配慮
