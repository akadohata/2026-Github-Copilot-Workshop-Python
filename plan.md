# Pomodoro Web App Implementation Plan

## フェーズ0: 土台
- Flaskで `GET /` 配信の最小構成
- `index.html` のDOM構造をモックに合わせて骨組み化
- `style.css` で大枠レイアウト（タイマー/コントロール/ダッシュボード枠）

## フェーズ1: タイマー最小機能（MVPコア）
- `state` と `timer` の基本ロジック実装
- 開始/一時停止/リセット
- モード切替（作業/短休憩/長休憩）
- カウントダウン表示（mm:ss）

## フェーズ2: サイクルと自動遷移
- 作業回数カウント
- 0秒到達時の自動遷移（短休憩/長休憩）
- UIラベル・状態の反映

## フェーズ3: アクセシビリティ要件
- マウスクリックで開始/停止
- キーボード操作（S:開始 / I:停止 / E:終了）
- フォーカス可視化など基本的な配慮

## フェーズ4: 設定と永続化
- 作業/休憩時間・長休憩回数の設定UI
- `localStorage` に保存
- 再読み込み時の復元

## フェーズ5: 履歴・可視化ダッシュボード（必須）
- セッション履歴の保存（作業回数、合計時間）
- ダッシュボード表示（簡易集計・トレンド）
- 必要なら日付単位の簡易集計

## フェーズ6: 仕上げ（任意）
- 音通知、ブラウザ通知
- タブタイトル残り時間表示
- レスポンシブ微調整
